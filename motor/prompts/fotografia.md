# Prompt exclusivo — FOTOGRAFIA / ESTÚDIO

> Estilo padrão: `foto-portfolio` (dark minimal, imagem manda). Tom autoral e
> emocional. Fotografia se vende pelo **portfólio** — as imagens são o herói.

## Personalidade
Sensível, autoral, elegante. Vende emoção e memória, não "serviço". Palavras-chave:
momentos, memória, olhar, registro, eternizar, história.

## Visual
- Dark e minimalista para a foto saltar. Pouco texto, muita imagem.
- O **portfólio** é a seção principal — galeria masonry filtrável por categoria.
- Fotos reais são essenciais: pedir/baixar do Instagram do fotógrafo.

## Seções
1. Hero — imagem forte + frase curta ("Eternize os seus melhores momentos").
2. Portfólio — galeria filtrável (casamento, ensaio, newborn, eventos).
3. Pacotes — casamento, ensaios, newborn, eventos (preço "a partir de" quando der).
4. Sobre — o fotógrafo, o olhar, a abordagem (opcional, curto).
5. Depoimentos — clientes emocionados (forte prova social), só se reais.
6. Contato — **orçamento** pelo WhatsApp, Instagram, cidade/atende em.

## Tom dos textos
Emocional e direto. Ex.: "Cada clique guarda uma história que você vai querer
reviver para sempre."

## data.json — campos-chave
- `portfolio.items`: cada item tem `cat` (casamento/ensaio/newborn/eventos),
  `titulo` e `image`. **Trocar os placeholders por fotos reais** do trabalho.
- `instagram` é vitrine — linkar o @ real (é onde o portfólio vive).
- `services` = pacotes; preço "A partir de R$ X" converte melhor que "sob consulta".

## ⚠️ Quando faltar contato/fotos no Maps
Fotógrafo quase sempre só tem Instagram. Buscar o @ e, se possível, fotos públicas
para o portfólio — sem elas, a demo perde a força (a imagem é o produto).
